SUMIF Function

Last updated: 10 January 2025 •
Home › Functions › Math › SUMIF Function

SUMIF adds up values in a range that meet a single condition. For multiple conditions, use SUMIFS instead.

Syntax

=SUMIF(range, criteria, [sum_range])

Arguments

Required arguments

range - The range of cells to evaluate against the criteria.

criteria - The condition that determines which cells to sum.

Optional arguments

sum_range - The actual cells to sum. If omitted, Excel sums the cells in range.

Examples

Sum sales for a specific product

=SUMIF(A2:A100, "Apple", B2:B100)

Sums all values in column B where column A equals "Apple".

Sum values greater than 100

=SUMIF(B2:B100, ">100")

Sums all values in column B that are greater than 100.

Common Errors

Returns 0 - The criteria doesn't match any cells. Check for typos or extra spaces.

#VALUE! - Usually caused by an invalid criteria format.
